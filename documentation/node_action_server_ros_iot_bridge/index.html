Module node_action_server_ros_iot_bridge

This module is an action server for iot bridge. It work subscribe to MQTT and perform actions on the basis of message.

Classes

This class receives the goals and process them. * self.on_goal - It is the function pointer which points to a function which will be called when the Action Server receives a Goal. * self.on_cancel - It is the function pointer which points to a function which will be called when the Action Server receives a Cancel Request.
